# Skills

Personal Agent Skills.

| Skill | Description | Install |
| --- | --- | --- |
| [chrome-e2e-qa-recorder](skills/chrome-e2e-qa-recorder/SKILL.md) | Run Chrome-based end-to-end QA flows and export video traces from Computer Use screenshots. | `npx skills add lordidiot/skills --skill chrome-e2e-qa-recorder` |
| [pet-fringe-cleanup](skills/pet-fringe-cleanup/SKILL.md) | Clean chroma-key fringe and halos from finalized hatch-pet spritesheets. | `npx skills add lordidiot/skills --skill pet-fringe-cleanup` |

## Install a Skill

```bash
npx skills add lordidiot/skills --skill <skill-name>
```
